What will the following code output and why? 

  var b = 1;
  function outer(){
        var b = 2
      function inner(){
          b++;
          var b = 3;
          console.log(b)
      }
      inner();
  }
    outer();

My answer: Looks like another hoisting question. var b = 2 would override var b = 1;. In the inner function, var b would be hoisted to the top and override the outer b, then b++ would do nothing because it's undefiend at that point until it gets assigned to 3, so the final log should be 3.

Their answer: Yea, and I guess their code makes it more clear what's going on in the inner function. 


  function inner () {
    var b; // b is undefined
    b++; // b is NaN
    b = 3; // b is 3
    console.log(b); // output "3"
  }
