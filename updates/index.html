Etch-A-Sketch Nov 5, 2019

Completed this project via The Odin Project.

(Click image to try it)

I had a bunch more ideas about how to add features, like creating different layers that get swapped via a queue so you could mix colors / undo things. Ultimately I learned some of the limits of JS when it comes to styling divs based on mouseover - much more than a seven-bit grid (need to find a font with numbers or pay for this one haha) of divs and it starts to get pretty laggy (at least on my computer).

Compared to the Rock-Paper-Scissors project, there were way fewer visual elements, and as a result way fewer hours / lines of code used for this one. Lesson learned for the future :).
